N-Queens Problem

How to play this game Play the game

1. Start with an empty NÃ—N chessboard.
2. Place one queen in each row, starting from the first row.
3. For each queen:
- Place it in an unoccupied square within its row.
- Ensure that no other queen shares the same column or diagonal with the current queen.
- If a conflict is found, try a different square in the same row.
- If all squares in the row are exhausted, backtrack to the previous queen and try a different square for it.
4. Repeat this process until all queens are successfully placed on the board without any conflicts.
5. If a valid placement of N queens is achieved, it represents a solution to the N-queens problem.
6. If it is not possible to place all queens without conflicts, then no solution exists for the given N.
